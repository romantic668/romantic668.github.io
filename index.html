Full-stack Apps

Bug Tracker

Bug tracker helps software engineers and web developers keep track of their weekly sprints in scrum development, and thus preventing over budget and overtime projects.

Features

Users are able to log in the system securely with JWT authentication. They are able to report and keep track of bugs in a real time fashion. They are able to update and delete bugs as needed.

Technologies

The front-end is built with React and Redux, bootstrap. The back-end is built with Node.js and Express and JWT for authentication. Database runs on Mongodb. Socket.io is used for real time implementation.

Future improvements

Users should be able to leave comments. Reopening bugs should be made available. Update and delete buttons should be greyed out when users are not authorized.

Trivia Game

Trivia game meets people's entertainment needs by providing users with real time communication and learning through play experince.

Features
